# springproject

Spring 5 버전을 이용한 웹에서 가장 기초적인 게시판을 구현한 프로젝트입니다

FrameWork : Spring 5.0.7

IDE : Eclipse

Database : Oracle 11g xe

Server : Tomcat 9

Java : jdk 1.8

웹 디자인 : Bootstrap

주요 구현 기능

1. 페이징 처리
2. Ajax와 REST처리 방식을 이용한 댓글 CRUD
3. MyBatis의 동적 SQL을 이용한 검색기능
4. 파일첨부
5. Spring Security를 이용한 로그인 및 권한 부여
